Hirenet - Decentralized Job Finder App

Hirenet is a full-stack decentralized job portal that connects job seekers and recruiters directly without intermediaries. It enables transparent, secure, and efficient job matching using modern web technologies.


---

🚀 Features

👤 For Job Seekers:

Register and create a professional profile

Browse available job listings

Apply for jobs directly

Track application status


🧑‍💼 For Employers:

Register as a recruiter

Post new job openings

View applications and applicant profiles

Hire and manage applicants directly


🌐 General:

User authentication (Login/Register)

Responsive and modern UI

Decentralized job listing model (Decentralization to be enhanced in future versions)

🧑‍💻 Setup Instructions

Prerequisites

Node.js

MongoDB

Angular CLI (if frontend is in Angular)


Clone the Repo

git clone https://github.com/yaxxhsri7444/Hirenet-Decentralized-Job-Finder-App.git
cd Hirenet-Decentralized-Job-Finder-App

Backend Setup

cd HireNet-Backend
npm install
npm start

Frontend Setup

cd ../HireNet-Frontend
npm install
ng serve
